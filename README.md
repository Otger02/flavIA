# Flavia

Base scaffold for a conversational intimate wellbeing platform built with Next.js, TypeScript and Tailwind CSS.

## Stack prepared

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- Vercel-ready structure

## Initial folders

- `app`
- `components`
- `features`
- `lib`
- `types`
- `sanity`
- `supabase`

## Available scripts

- `npm run dev`
- `npm run build`
- `npm run lint`

## Pending integrations

The project intentionally does not include business logic yet. Supabase, Stripe and Sanity runtime dependencies can be installed in a later step when integration work starts.
